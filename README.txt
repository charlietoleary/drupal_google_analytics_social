-- DRUPAL MODULE --

Name: Google Analytics Social Tracking
Author: Charlie O'Leary <http://drupal.org/user/948772>
Drupal: 7
Release Status: This module is in very early development, and is in need of comprehensive review, testing and new features.

-- DESCRIPTION --

A submodule for the Google Analytics Module to enable social media tracking.

This module allows your drupal website to utilize the new feature of Google Analytics: "Social Interaction Analytics".

Once installed, your Google Analytics profile should now register social actions
such as Facebook "Likes" & Twitter "Tweets".

-- INSTALLATION --

Required modules: googleanalytics

-- CONFIGURATION --

Configure settings in admin/config/system/googleanalytics.
